# Coolibreat | Eventos & Experiencia

En **Coolibreat**, cada evento es una oportunidad para crear algo extraordinario. Una productora con base en **Esquel, Chubut**, dedicada a diseñar experiencias que trascienden lo convencional. 

Combinando curaduría artística, la energía de la música electrónica y la excelencia en hospitalidad para crear momentos que quedan grabados en la memoria.

## 🛠️ Stack Tecnológico
* **Vite + React** & **TypeScript**: Para una interfaz rápida y robusta.
* **Tailwind CSS**: Estética minimalista y responsive.
* **EmailJS**: Integración fluida para la gestión de nuevas consultas y proyectos.

## 📂 Estructura del Repositorio
* `/public/assets`: Galería de eventos, marcas y assets visuales.
* `/src`: Componentes modulares de la interfaz.
* `tailwind.config.js`: Personalización de la identidad visual de la marca.
